# 🛍️ ShopForge

A full-stack e-commerce platform built with FastAPI, MongoDB, and React.

## 🚀 Features

- JWT Authentication
- Product Search & Filtering
- Shopping Cart
- Order Management
- Admin Dashboard
- MongoDB Integration
- Responsive UI
- REST APIs with FastAPI

## 🛠️ Tech Stack

Frontend
- React 18
- Vite
- Tailwind CSS
- Zustand
- React Router

Backend
- FastAPI
- MongoDB
- Motor
- Pydantic v2
- JWT Authentication

## 📂 Project Structure

text ShopForge ├── backend │   ├── app │   ├── main.py │   └── requirements.txt └── frontend     ├── src     └── package.json 

## ⚙️ Setup

### Backend

bash cd backend python3.12 -m venv venv source venv/bin/activate pip install -r requirements.txt uvicorn main:app --reload 

### Frontend

bash cd frontend npm install npm run dev
